# Research Skill

Use this skill when you need to find up-to-date information, documentation, or code examples for any library, framework, or architectural pattern.

Instructions:
1. Identify the core technology or pattern to research.
2. Use the `context7` MCP server to query for relevant documentation.
3. Summarize findings and provide actionable recommendations.
